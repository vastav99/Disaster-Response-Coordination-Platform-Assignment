# Disaster Coordination Platform

This is a Node.js + Supabase-based backend with simple frontend for testing.

## How to Run

1. `cd backend`
2. `npm install`
3. `npm start`

Frontend: Open `frontend/index.html`

## API Endpoints

- `POST /disasters`
- `GET /disasters?tag=`
- `GET /resources/:id/resources`
- `POST /verify-image`

Mock integrations are in `backend/mock/`.
